# Prueba Técnica AdsProject Media: Ruleta Interactiva

Este proyecto es una prueba técnica desarrollada para la agencia AdsProject Media. Consiste en una ruleta interactiva implementada utilizando HTML, SCSS y JavaScript. La ruleta es completamente funcional y se adapta a diferentes tamaños de pantalla, garantizando una experiencia de usuario óptima.

## Características Principales

Interactividad: La ruleta permite al usuario interactuar con ella, realizando un giro al presionar un botón.

100% Funcional: La ruleta está programada para seleccionar una opción aleatoria cada vez que se realiza un giro.

100% Responsive: Se ha implementado un diseño completamente responsive que se adapta a diferentes dispositivos y tamaños de pantalla.

## Tecnologías Utilizadas

HTML5: Utilizado para la estructura y marcado semántico del proyecto.

SCSS: Un preprocesador de CSS que facilita la organización y mantenimiento del estilo del proyecto.

JavaScript Vanilla: Utilizado para la interactividad y funcionalidad de la ruleta.

## Instrucciones de Uso

Clona el repositorio en tu máquina local.

Abre el archivo index.html en tu navegador web preferido.

En la página, encontrarás una ruleta y un botón para girarla.

Haz clic en el botón "Girar" para iniciar el giro de la ruleta y observa cómo selecciona una opción aleatoria.

O visita [prueba-adsproject.web.app](prueba-adsproject.web.app)

## Estructura del Proyecto

index.html: Archivo principal que contiene la estructura HTML del proyecto.

css/style.scss: Archivo SCSS que contiene los estilos para el proyecto.

js/main.js: Archivo JavaScript que contiene la lógica de la ruleta.

img/: Carpeta que contiene las imágenes utilizadas en el proyecto.
